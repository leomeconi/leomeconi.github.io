# https://leomeconi.github.io/

For this final project I created a very clean website for my band with some infos about us, links to our music and social media, and photos.

I started from the HTML file we created in class with our name. 
From that I took the code we used in class during the band project and started modifying with my texts.
First off I changed the basic stuff, like the title, the sections, and I also cleaned the code as much as I could to start working on it in the simplest way possible. 



### Organizing the page

The layout of the page is very basic but, as I said in my project proposal, I wanted it to be minimal and clean, to put just the essential things in it.
