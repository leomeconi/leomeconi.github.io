# https://leomeconi.github.io/

For this final project I created a very clean website for my band with some infos about us, links to our music and social media, and photos.

I started from the HTML file we created in class with our name. 
From that I took the code we used in class during the band project and started modifying with my texts.
First off I changed the basic stuff, like the title, the sections, and I also cleaned the code as much as I could to start working on it in the simplest way possible. 



### Organizing the page

The layout of the page is very basic but, as I said in my project proposal, I wanted it to be minimal and clean, just to put the essential things in it. 

I remember the first class of this semester the professor suggesting not to use LLM for the entire semester and "save" the use of that for the final project... that's exactly what I did.
For this project I tried so hard not to use AI for the first part of it but then I needed for a lot of things that I really didn't know. 

For the first part of the work I used ChatGPT, but then I switched to Claude. 
I actually made a mistake and realized about that half way through the process, that for the interaction with ChatGPT I asked everything in Italian. 
I'm going to summarize everything I wrote after each screenshot btw.
